# skaM
### P2P-мессенджер для локальных сетей / P2P messenger for local networks

---

**Русский**

## О проекте

**skaM** — это P2P-мессенджер, который работает через локальную сеть (LAN, Radmin VPN, ZeroTier, Hamachi). Никаких серверов. Сообщения идут напрямую между компьютерами.

Название — игра слов. Макс наоборот.

## Почему это работает там, где другие нет

Обычные мессенджеры имеют серверы. Серверы можно заблокировать. У skaM нет серверов — значит, нечего блокировать. Ваш трафик идет внутри зашифрованного туннеля локальной сети. РКН его просто не видит.

## Возможности

- P2P соединение без центральных серверов
- Не требует регистрации и номера телефона
- Работает через Radmin VPN / ZeroTier / LAN
- Сообщения идут напрямую между компьютерами
- Не хранит ваши данные на чужих серверах
- Простота: указал IP друга — общаешься

## Как начать

1. Установите программу для создания локальной сети и настройте её:
   - Radmin VPN
   - ZeroTier
   - Hamachi
   - Или просто подключитесь к одному Wi-Fi
2. Скачайте skaM
3. Запустите
4. Введите IP адрес собеседника
5. Начинайте общение
